# Configurer l'environnement virtuel et ajouter les dépendances nécessaires

`chmod 774 ./configure`

`./configure`

# Lancer le serveur Flask et l'interface web

`chmod 774 ./run`

`./run`

L'interface est accessible à `http://localhost:8000`
